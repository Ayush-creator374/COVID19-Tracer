# COVID19-Tracer
This is a covid19 tracer implemented using web scraping

## Run Code
```
git clone https://github.com/Shivanshu10/COVID19-Tracer.git
cd COVID19-Tracer
pip install requirements.txt
python3 code.py
```

## Compile Code
```
git clone https://github.com/Shivanshu10/COVID19-Tracer.git
cd COVID19-Tracer
pip install requirements.txt
pyinstaller code.py --onefile --additional-hooks-dir=.
cd dist
./code
```
